Em \colorurl{https://github.com/j-Lago/pyTicTacToe} você encontrá o repositório de um código python para uma
interface gráfica do jogo da questão anterior.
Ao executar \colorurl{main.py}, o programa abrirá uma janela gráfica com um tabuleiro tic-tac-toe vazio.
Ao clicar com o botão esquerdo do mouse em uma das posições do tabuleiro, o programa chamará a função
\inlcode{eval\_move()} do arquivo \colorurl{eval_move.py} (o arquivo existe, mas a implementação
da função está faltando).

Edite o arquivo \colorurl{eval\_move.py}, incluindo a sua implementação para a função desenvolvida na
questão anterior, execute \colorurl{main.py} e verifique se sua função realmente implementa a lógica do jogo.
